Method 2

Name: Searching for a String in a String
Description:This method returns the position of the first occurrence of a specified text in a string.
Example using it:
const string = "Usman is a great teacher!";
let position = string.search("great");
console.log(position);
it will return 11

Method 3

Name: Converting a String to an Array
Description: Any string can be converted to an array with the split() method.
Example using it:

let string = "Mansour , Ali , Ahmed , Abdul"; 
let stringArray = string.split(","); 
         
console.log(stringArray[1])

it will show Ali.
